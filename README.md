# 🌸 Maison Elise — Luxury Boutique Landing Page

A modern and elegant **Boutique Store Landing Page** built using **HTML, CSS, Bootstrap 5, and JavaScript**.  
This project delivers a **premium user experience**, smooth animations, and a clean luxury design suitable for showcasing products and attracting clients.

---

## 🚀 Live Preview
🔗 [View Live Demo](https://maison-elise-boutique.netlify.app)

---

## ✨ Features

- 🎨 Modern and luxury UI design  
- 📱 Fully responsive (mobile-first approach)  
- 🧭 Sticky navigation bar  
- 🛍️ Product showcase / gallery section  
- 🎯 Call-to-action buttons (Shop Now)  
- 🌟 Smooth scroll animations  
- 🧩 Clean layout with Bootstrap grid system  
- 🖼️ High-quality product and hero images  
- 📩 Newsletter / contact section  
- 🔗 Social media links
